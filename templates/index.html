Satara Tourism API

If you want to add Tourism information about satara to your website or app then you've come to the right place!

This Application Programming Interface (API) allows a web or mobile developer to use the Tourism data in their web or mobile application(s).

I created a public, fast and clean REST API that's easy to use, returns JSON.I gathered all destination data from different website.

I created this API for learning purpose.
